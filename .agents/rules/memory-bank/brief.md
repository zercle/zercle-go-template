# Project Brief: Zercle Go Fiber Template

A production-ready Go microservice template designed for high performance and scalability. It leverages the Fiber framework, Clean Architecture principles, Domain-Driven Design (DDD), and PostgreSQL with UUIDv7 support.

**Key Goals:**
- Provide a robust starting point for Go microservices.
- Ensure type safety and maintainability through Clean Architecture.
- Standardized logging, configuration, and error handling.
